# 서비스 흐름도

⚠️ IntelliJ IDE 의 Mermaid 표시 버그로 한글 표시가 안 됨. Github 에서 확인해 주세요.

```mermaid
flowchart TD
    Start[시작] --> A1(게임상태를 초기 상태로 설정한다)
    A1 --> C1{게임상태가 종료상태가 아닌가?}
    C1 -->|예| A2(수비 플레이어가 숫자를 생성한다)
    A2 --> A3(공격 플레이어 숫자 입력 메시지를 화면에 출력한다)
    A3 --> A4(공격 플레이어로부터 숫자를 입력 받는다) 
    A4 --> C2{입력값이 유효한 값인가?}
    C2 -->|예| A5(수비 플레이어가 숫자를 비교하여 판정한다)
    C2 ------->|아니요| A6(예외를 발생한다)
    A6 --> End
    A5 --> A7(결과를 출력한다)
    A7 --> C3{결과가 스트라이크 인가?}
    C3 -->|예| A8(축하메시지를 화면에 출력한다)
    C3 -->|아니요| A3
    A8 --> A9(게임 진행 여부 메시지를 화면에 출력한다)
    A9 --> A10(공격 플레이어로부터 진행 여부를 입력 받는다)
    A10 --> A11(진행여부를 게임상태에 반영한다)
    A11 --> C1
    C1 ------>|아니요| End[종료]
```
